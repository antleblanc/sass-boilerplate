## Architecture

The [7-1 Pattern](http://sass-guidelin.es/#the-7-1-pattern) up 9-1 Pattern.

```sh
stylesheets/
├── base/
├── components/
├── layout/
├── modules/
├── pages/
├── shame/
├── themes/
├── utils/
└── vendor/
```

Two additions with `modules` and `shame` folders.
